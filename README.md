# SVM-Softmax-Classifier-CIFAR10-dataset also using KNN

### Required libraries 
- Numpy
- scipy
- Keras
- matplotlib

### Running the SVM program
```
python runSvmSoftmax.py
```


### Running the KNN program
```
PYTHON runKnnSoftmax.py
```
